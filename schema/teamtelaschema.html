Team TELA Schema

Material: Card stock

Fastener: Slot and tab

Theme: Animal

Design: Option 2 (3D Style)

Rules: Shapes with a single tab on one side can connect to any slot. Shapes with more than one tab on one side can only connect to a shape with an equal amount of slots.

Roles: Each member will create a set of body parts. For example, one member will create a set of varying legs while another member will create a set of varying heads.

Members will be able to attach their shapes onto other member’s shapes in hopes of creating an animal-like form.

The strength of this construction system is the ability to create both flexible and sturdy connections. The aesthetic will be organic geometric shapes meaning the shapes that we create will have rounded edges and little to no sharp corners.

First Fastener Design:

This is the old fastener design. This design left too much extraneous material that affected the overall look of the combined shapes which made it look messy and not cohesive.

This is the first laser cutout test the group did for our first fastener design.

Second Fastener Design:

This fastener design is the most recent. This fastener design is flexible and can be made sturdy if combined with multiple fasteners. An example of this can be found in the “How it works” section further down below.

How it works using our second fastener design:

Tabs and slots will slide into place and a second set of tabs and slots will secure it in place.

Some shapes will have multiple tabs or slots that only fit to certain shapes. Shapes with multiple tabs and slots will be a lot more sturdy and stiff while shapes with a single connection of tabs and slots will be more flexible.
